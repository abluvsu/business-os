## ⚡ Core Loop (The Only Thing You Memorize)

```
1. OPEN GAME (2 min)
   → Check Energy. < 2 hearts? → Close game. Sleep. No guilt.

2. PICK QUEST (1 min)
   → "Today I feel like... [Roast Request / Manifesto / Nothing]"

3. DO ONE MICRO-TASK (15–30 min)
   → Timer starts. Phone away. One tab only.
   → Task = "Write 3 sentences" / "Record 10 sec video" / "Push 1 file"

4. SAVE ARTIFACT (2 min)
   → Screenshot / PDF / Git commit / Voice memo.
   → Name it: `ART-001-manifesto-draft-1.md`

5. LOG XP (30 sec)
   → Open `SAVEFILE.md` → Add line: `+10 XP | ART-001 | Manifesto draft 1`
   → Close file.

6. CLOSE GAME
   → Energy -1 heart. Sanity unchanged (or +1 if it felt fun).
   → Real life resumes.
```

**That's it. No dashboards. No metrics. No "did it work?"**

## 💾 Save System (Failure-Proof by Design)

| Situation | What Happens |
|-----------|--------------|
| **Missed 2 weeks?** | Save file waits. No decay. No "streak broken." |
| **Tweet flopped (0 likes)?** | XP still granted. Artifact saved. `ART-047` exists. |
| **Roast request got 0 submissions?** | You still have `ART-012-empty-form.md`. XP granted. |
| **Felt stupid recording video?** | `ART-003-cringe-take-1.mp4` saved. XP granted. Delete later if you want. |
| **Energy = 0 for a month?** | Character sleeps. World pauses. You're still Level 3 when you return. |

**There is no Game Over. Only Pause.**
